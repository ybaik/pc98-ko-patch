## PC98용 초시공요새 마크로스: 리멤버 미 한글패치
### 정보
* 최종 수정일: 2026.01.10
* 프로그램, 번역: 천수도인
* 프로그램, 그래픽: 낭만엘리
* 제작자 블로그: https://hyunxx.tistory.com/

DOS버전을 참고하려 했지만, Macross2, 3와 같이 결국 그냥 새로 한글화를 했습니다.
DOS버전과는 다르게 일본이름 그대로를 유지하려 하였습니다.
각 명칭의 한글 발음들은 나무위키를 참조하였습니다.

패치 방법
* 우선 patch 폴더 내 파일을 덮어 쓰신 후, patch-expand 폴더 내 파일을 덮어 쓰시면 됩니다.

패치하시면 다음의 조건으로 HDD모드로 작동 가능하실 겁니다.
- 기본 설치된 파일들이 'a:'에 있다고 가정.
- Anex86, neko project 지원을 위한 b:기준 패치는 patch-b-drive 폴더의 MAC.EXE를 사용하시기 바랍니다.
