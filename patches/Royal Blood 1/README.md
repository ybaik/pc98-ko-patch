### Patch 방법
* 각 패치 파일을 다음의 장소로 넣어주세요.
  - MAIN.EXE -> disk 1
  - OPEN.EXE, RBDEND.EXE, OPEN.DAT -> disk 2

### 플레이 방법
* PC98 에뮬레이터 1번 디스크에 Disk1, 2번 디스크에 Disk2를 넣어줍니다.
* 제공 되는 안내에 따로 2번 디스크를 유저디스크로 바꿔줍니다.
* 유저디스크를 새로 작성하시면 Disk1과 유저디스크만으로 바로 게임이 가능합니다.
